# 🌙 睡眠

> 为什么经前更容易睡不好，以及可以怎么做

---

## 为什么经前睡不好

经前，**雌激素和孕酮双双下跌**，这波激素退潮会从多个方面打扰你的睡眠。

- 📉 **孕酮下降**：排卵后升高的孕酮本让人犯困，经前它一回落，困意也随之消失
- 🌗 **褪黑素受抑制**：激素骤降会压低"助眠激素"褪黑素，导致入睡难、睡得浅
- 🌡️ **体温偏高**：排卵后体温略升，而入睡需要体温下降，因此更难睡着
- 💗 **血清素波动**：经前血清素不足，会带来情绪低落、疲劳与睡眠问题

> ⏰ 睡眠和情绪最糟的窗口，通常在**月经来潮前 4~5 天到来潮前 2 天**。

## ✅ 改善睡眠的方法

| 做法 | 说明 |
|---|---|
| ☕ **控制刺激物** | 避免咖啡因、酒精和高糖食物，尤其下午之后 |
| 🍗 **吃对食物** | 多吃蛋白质与复合碳水，帮助情绪与睡眠稳定 |
| 🏃 **注意运动时机** | 白天运动好，但**睡前不要剧烈运动**（会升高体温） |
| 🌡️ **营造凉爽环境** | 卧室偏凉、睡前温水澡后让体温自然下降 |
| 🧘 **减压** | 长期压力升高皮质醇 → 抑制褪黑素，形成恶性循环，减压能打破它 |
| 🌙 **固定作息** | 规律的上床/起床时间，帮助稳定生物钟 |

## 🚩 什么时候需要就医

如果失眠**严重或持续**，或伴随其他异常，建议就医：
- 可能需要排查甲状腺功能、多囊卵巢综合征(PCOS)等
- 医生也可能在医生指导下考虑激素或药物方案
- 可用量表（如 PSQI、ISI）量化睡眠问题

👉 见《什么时候可以考虑寻求专业帮助》。
