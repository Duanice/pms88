# 💗 什么时候可以考虑寻求专业帮助

> 识别需要关注的信号，知道何时向专业人士求助。

---

## 先说最重要的

大多数经前不适会随月经开始而缓解，属于正常范围。
但当症状**严重、持续恶化，或影响到你的正常生活**时，寻求专业帮助是明智且值得的——
**这不是小题大做，而是照顾自己。**

## ⚠️ 建议预约就医的信号

只要符合其中一条，就值得找专业人士聊聊：

| 信号 | 说明 |
|---|---|
| 📉 **影响生活** | 经前症状干扰到工作、学习、社交或家庭关系 |
| 📈 **逐月加重** | 症状随时间越来越严重，而不是保持稳定 |
| 😔 **情绪很重** | 显著的抑郁、焦虑、易怒或情绪不稳，难以自行调节 |
| 🔁 **规律但强烈** | 每个月经前都被严重情绪问题困扰（可能是 PMDD） |
| 🤕 **躯体难忍** | 腹痛/头痛等严重到药物无效或需要请假 |
| 🌫️ **持续不退** | 症状月经后仍不消失（可能不只是经前问题） |

## 🏥 应该看哪个科？

PMS / PMDD 的诊疗**同时涉及妇科和精神科**：

```
   你的主要困扰是……
        │
   ┌────┴─────┐
   ▼          ▼
🩺 身体为主   🧠 情绪为主
(腹痛/乳房胀/  (抑郁/焦虑/
 月经异常)     易怒/失眠)
   │            │
 妇科 ────────► 需要时转介 ──► 精神科 / 心理健康科
```

- **妇科**：通常是首诊科室，负责确诊并排查其他疾病
- **精神科 / 心理健康科**：当情绪症状突出时会转介，做正式评估与治疗

---

## 🎒 就医前的一个小准备

带上你的**症状记录（至少 2 个月经周期）**会非常有帮助。
医生常常通过"症状出现和消退的时间点"来判断：
- 若症状**经前出现、来潮后消退** → 更可能是 PMS / PMDD
- 若症状**全程持续** → 可能是独立的抑郁或焦虑，需要不同处理

> 👉 参见《如何记录规律，但不过度紧张》学习怎么记。

---

## 🌱 求助不是软弱

PMDD 等是**被正式承认、可以被识别和管理的疾病**，也常常被漏诊。
主动求助，是把"判断严重性"的工作交给专业人士——你值得被认真对待。
